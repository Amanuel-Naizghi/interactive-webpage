# Interactive Webpage Project

## Overview
This project focuses on building an interactive webpage using **HTML** and **CSS Flexbox**. The goal of the assignment is to practice modern web layout techniques and understand how flexible layouts can be used to create responsive and organized web pages.

## Objectives
- Create a structured webpage using **HTML**.
- Style the webpage using **CSS**.
- Use **Flexbox** to build flexible and responsive layouts.
- Organize webpage content such as headers, sections, navigation, and images.
- Improve front-end development skills through practical implementation.

## Features
- Structured HTML page layout
- Responsive design using **CSS Flexbox**
- Organized content sections
- Basic webpage styling and formatting
- Interactive layout elements

## Technologies Used
- HTML5
- CSS3
- CSS Flexbox

## Learning Outcomes
Through this project, we learned how to structure webpages using semantic HTML elements and how to use Flexbox to control alignment, spacing, and responsiveness in modern web design.
